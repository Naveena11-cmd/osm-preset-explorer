# OSM Preset Explorer Prototype

## Overview : 
This project is a prototype web application developed to explore and visualize tagging presets from the OpenStreetMap tagging schema. The application provides an interactive interface to browse, search, and inspect preset metadata such as tags, geometry types, fields, and inheritance relationships.
This prototype was created as preparation for contributing to the OpenStreetMap ecosystem and as part of a Google Summer of Code (GSoC) proposal.

---

## Features : 
- Load preset data from the OpenStreetMap tagging schema
- Search presets by name or identifier
- Display preset details including tags and geometry
- View inheritance relationships between presets
- Show regional availability information
- Interactive user interface with sidebar navigation

## How It Works : 
The application fetches preset and field data directly from the OpenStreetMap tagging schema repository and displays the data dynamically in a browser interface.

The system performs the following steps:
1. Fetch preset data
2. Store data in memory
3. Render preset list
4. Display selected preset details
5. Allow searching and filtering
